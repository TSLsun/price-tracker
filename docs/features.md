# 功能特性

Price Tracker 提供了多樣化的功能來優化您的購物體驗。

## 1. 自動價格追蹤 (Automated Scraping)
*   **多平台支持**：支持 momo 購物網與 PChome 24h 購物。
*   **隱身技術**：採用 `playwright-stealth` 技術，模擬真用戶行為，確保抓取穩定性。
*   **極簡追蹤流程**：採用「連結優先」設計，只需貼上商品 URL 即可一鍵追蹤，其他欄位（如名稱、分類）皆為選填。

## 2. 智慧 Dashboard (Smart Dashboard)
*   **行內分類管理**：在新增或編輯商品時，可直接「行內新增」新分類，無需中斷流程跑去管理頁面。
*   **最划算標籤 (Best Deal)**：系統會自動計算同一分類下所有商品的單位價格，並自動為單價最低的商品標記「最划算」標籤。
*   **平均單價計算 (Unit Price)**：只需填入商品規格（如：500ml、30片），系統自動計算單位價格，方便不同包裝間的比價。
*   **搜尋與過濾**：快速查找特定商品或特定分類。

## 3. UX 體驗與移動端優化
*   **行動裝置友善**：按鈕大小與間距經過優化，刪除與編輯操作在手機上同樣直覺。
*   **即時反饋**：新增商品時有透明的 Loading 狀態與結果即時呈現。

## 4. LINE 降價通知
*   **精準監測**：每日背景巡檢，偵測到價格低於上次紀錄時立即發送通知。
*   **詳盡訊息**：通知包含商品名稱、舊價格、新價格以及直接購買連結。

## 5. 數據視覺化
*   **價格歷史**：完整紀錄每次巡檢的價格，方便觀察商品是否有「先漲後降」的現象。
