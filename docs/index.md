# Price Tracker 價格追蹤工具

歡迎使用 **Price Tracker**，這是一個專門為台灣電商平台（momo、PChome）設計的價格追蹤與比價工具。

## 專案緣起
在台灣網購時，價格波動大且平台眾多，手動比價既費時又費力。本專案透過自動化爬蟲、即時 UI 反饋與降價通知，幫助用戶在最佳時機下單。

## 核心價值
*   **極速追蹤**：「連結優先」設計，貼上 URL 即可追蹤，省去繁瑣輸入。
*   **智慧比價**：支援「單位價格」自動計算，系統會自動標記分類中「最划算」的商品。
*   **靈活管理**：支援行內快速新增分類，管理商品結構不中斷。
*   **即時通知**：價格下跌立即透過 LINE 通知，不再與優惠擦身而過。
