# Price Tracker 價格追蹤工具

歡迎使用 **Price Tracker**，這是一個專門為台灣電商平台（momo、PChome）設計的價格追蹤與比價工具。

## 專案緣起
在台灣網購時，價格波動大且平台眾多，手動比價既費時又費力。本專案透過自動化爬蟲與即時通知，幫助用戶在最佳時機下單。

## 核心價值
*   **省時**：自動監控價格，不再需要每天刷網頁。
*   **精準**：支持「單位價格」計算，一眼看出哪種包裝最划算。
*   **即時**：價格下跌立即透過 LINE 通知，不再與優惠擦身而過。
